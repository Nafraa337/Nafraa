Cara Order Top Up Mobile Legends:

  1. Masukkan **ID Pengguna MLBB Anda** dan **ID Server** dengan benar.
  2. Pilih nominal Diamond atau jenis membership/paket.
  3. Pilih metode pembayaran yang Anda inginkan.
  4. Klik tombol "Bayar Sekarang!".
  5. Anda akan diarahkan ke WhatsApp kami dengan detail pesanan yang sudah terisi otomatis.
  6. Admin kami akan segera merespons untuk **memverifikasi ID Anda** dan menginformasikan instruksi pembayaran.
  7. Setelah pembayaran dikonfirmasi dan ID terverifikasi, Diamond akan segera masuk ke akun game Anda!
  8. **Penting:** Kesalahan penulisan ID adalah tanggung jawab pembeli.
